## Proje Hakkında

OpenCV kütüphanesi kullanılarak geliştirilmiş, fotoğrafların yüklenip çeşitli işlemlerden geçirildiği görüntü işleme uygulaması. Kullanıcılar fotoğraflarını yükleyerek farklı filtreler ve dönüşümler uygulayabilir.

## Özellikler

- **Fotoğraf Yükleme:** Yerel dosyalardan görüntü yükleme
- **Görüntü İşleme:** OpenCV algoritmaları ile işleme
- **Filtreleme:** Çeşitli görsel efektler ve filtreler
- **Boyutlandırma:** Fotoğraf ölçeklendirme ve kırpma
- **Format Dönüşümü:** Farklı görüntü formatları arası dönüşüm

## Teknolojiler

- Python
- OpenCV
- NumPy
- Image Processing Algorithms
- Computer Vision

## Görüntü İşleme Özellikleri

- Renk uzayı dönüşümleri (RGB, Grayscale, HSV)
- Kenar algılama (Edge Detection)
- Gürültü azaltma (Noise Reduction)
- Histogram eşitleme
- Morfolojik işlemler

## Kullanım Alanları

- Fotoğraf düzenleme
- Görüntü iyileştirme
- Özellik çıkarma
- Ön işleme (Pre-processing)
