## Proje Hakkında

TradingView platformundan kripto para coin yorumları, beğeni sayıları, görüşleri ve kullanıcı adlarını otomatik olarak çekerek veritabanına kaydeden Python uygulaması.

## Özellikler

- **Web Scraping:** TradingView'den veri çekme
- **Veri İşleme:** Yorum, beğeni ve kullanıcı verilerinin işlenmesi
- **Veritabanı:** Çekilen verilerin sistematik şekilde saklanması
- **Otomasyon:** Otomatik veri toplama ve güncelleme

## Teknolojiler

- Python
- Web Scraping Libraries
- Database Management
- Data Processing

## Kullanım Alanları

- Kripto para piyasa analizi
- Sosyal sentiment analizi
- Kullanıcı davranış analizi
- Trend takibi
